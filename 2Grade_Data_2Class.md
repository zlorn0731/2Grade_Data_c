2장 : 배열과 구조체
# 배열   | array
# 구조체 | struct
- 한꺼번에 많은 자료를 표현해야 하는 경우

## 배열 | array
- 같은 형의 변수를 여러 개 만드는 경우에 사용
  --여러 개의 변수 선언 : int A0, A1, A2, A3, A4, A5;
--> 하나의 배열 선언 : int A[6];
### (예시)
```
[a3] [a5] [a6] [a2] [a4] [a1] <-- 변수 선언
{A[0]} {A[1]} {A[2]} {A[3]} {A[4]} {A[4]} {A[5]} <-- 배열 선언
```

## 배열의 특징
- 배열 : <인덱스, 요소> 쌍의 집합
  -- 인덱스가 주어지면 해당되는 요소가 대응되는 구조
- 배열은 동일한 이름으로 접근
  -- 변수는 각각 다른 이름으로 접근
### (예시)
```
int A0, A1, A2, A3, A4, A5;
```
- 만약 배열이 없다면?
  -- 반복문을 사용할 수 없다 : 코드의 길이
### (예시)
```
for(i = 0; i < 45; i++) {
   sum += A[i];
}
```
- 직접 접근(direct access) 방식
  -- 항목 접근의 시간 복잡도 : O(1) --> [√][][√][][][√][] : 차례대로 거쳐 가지 않아도 됨
- 순차 접근(sequential access) 방식
  -- 항목 접근의 시간 복잡도 : O(n) --> [√][√][√][][][][] : 차례대로 거쳐 가야됨

## 1차원 배열
- 선언 방법
  -- 자료형 배열이름[배열의 크기];
### (예시) int A[6];
```
| A[0] |  <-- base : 배열 A의 주소    |
| A[1] | <-- base + sizeof(int)      |
| A[2] | <-- base + 2 * sizeof(int)  |
| A[3] | <-- base + 3 * sizeof(int)  |
| A[4] | <-- base + 4 * sizeof(int)  |
| A[5] | <-- base + 5 * sizeof(int)  |
- sizeof() : 자료형의 크기를 반환하는 연산자 --> (예시) sizeof(int);, sizeof(x);
```
  -- 모든 요소들이 메모리의 연속된 공간에 저장된다

- 배열의 초기화
### (예시)
```
int A[6] = {1, 2, 3, 4};
[0] -> 1
[1] -> 2
[2] -> 3
[3] -> 4
[4] -> 0
[5] -> 0
```
- 배열은 대입연산자를 이용해 복사 불가
### (예시)
```
int A[4], B[4] = {1, 2, 3, 4};
A = B; // 오류

- 올바른 복사
1번째 
A[0] = B[0];
A[1] = B[1];
A[2] = B[2];
A[3] = B[3];

2번째
for(i=0; i<4; i++)
     A[i] = B[i];
```

- 배열의 복사
### (예시)
```
변수의 복사와 배열의 복사
#include <stdio.h>
int main()
{
	int A[5] = { 10,20,30 };
	int B[5], i, x = 2018, y = 0;
	y = x;
	for (i = 0; i < 5; i++)
		B[i] = A[i];
	printf("변수 복사 결과: x=%d, y=%d\n", x, y);
	printf("배열 복사 결과: \n");
	for (i = 0; i < 5; i++)
	{
		printf("A[%d] = %d\t", i, A[i]);
		printf("B[%d]=%d\n", i, B[i]);
	}
}

- 결과
변수 복사 결과: x=2018, y=2018
배열 복사 결과:
A[0] = 10   B[0] = 10
A[1] = 20   B[1] = 20
A[2] = 30   B[2] = 30
A[3] = 0    B[3] = 0
A[4] = 0    B[3] = 0
```
